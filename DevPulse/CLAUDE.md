# DevPulse CLAUDE Reference

This repository includes custom CLAUDE command support and project documentation for the DevPulse capstone.

## Available Commands
- `run-backend` — Start the backend API from the `Backend/DevPulse.API` project.
- `run-frontend` — Start the frontend development server from `Frontend`.
- `run-tests` — Run the backend unit test suite and collect coverage results.

## Project Summary
DevPulse is a developer analytics dashboard with GitHub repository connectivity, JWT-based authentication, repository and metrics management, and a React-based UI.

## Key Deliverables
- `Docs/SPEC.md` — full project specification and development phases
- `Docs/API.md` — API endpoint documentation
- `Docs/SECURITY-AUDIT.md` — security audit report
- `.github/workflows/ci.yml` — CI pipeline configuration
- `.mcp.json` — MCP server integration manifest
- `.claude/commands/` — custom command definitions
